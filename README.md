# HTL-Medical

## Getting started

#### Clone the repo
```
git clone https://github.com/hoale0231/HTL-Medical.git
cd HTL-Medical
```

#### Install dependencies
For server machine
```
cd Server
pip install -r requirements.txt
```

For client machine
```
cd FrontEnd
npm install
```

## Run the app:
On the server, run:
```
cd Server
python app.py
```

For the client webapp, run:
```
cd FrontEnd
npm start
```
